## :truck: **RUNNING**

```shell
# Clone repository
$ git clone git@github.com:EpitechPromo2026/B-YEP-400-REN-4-1-zappy-tristan.bros.git
$ cd B-YEP-400-REN-4-1-zappy-tristan.bros

# Dependencies
$ make

# Run Zappy Server
$ ./zappy_server –help
```
